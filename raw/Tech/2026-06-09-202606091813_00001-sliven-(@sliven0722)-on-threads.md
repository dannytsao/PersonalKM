## 摘要
OSIRIS 是一個開源軟體專案，整合了全球飛機、衛星、船舶動態及災害事件等多種即時資訊，並提供 WHOIS 查詢、DNS 分析、Port Scanner 等多種 OSINT 工具。使用者可以像使用 Google Maps 一樣探索這些資料，且可透過 Docker 快速部署，且多數功能無需申請 API Key。

## 重點
- OSIRIS 是一個開源軟體專案，整合了全球飛機、衛星、船舶動態及災害事件等多種即時資訊，並提供 WHOIS 查詢、DNS 分析、Port Scanner 等多種 OSINT 工具。使用者可以像使用 Google Maps 一樣探索這些資料，且可透過 Docker 快速部署，且多數功能無需申請 API Key。

## 原始內容
Sliven (@sliven0722) on Threads 你有沒有想過， 以前只有政府、軍方或大型情報機構才看得到的東西， 現在有人直接做成開源軟體放到 GitHub 上。 這個專案叫做 OSIRIS。 打開之後，你會看到一顆可以自由旋轉的 3D 地球。 然後開始懷疑自己是不是誤闖了某個情報中心。 因為同一個畫面裡，你可以看到： ✈️ 全球上萬架飛機即時位置 🛰️ 數千顆衛星運行軌跡 🚢 海上船舶動態 🔥 火災與地震事件 📹 公開 CCTV 畫面 ⚠️ 全球衝突地區狀況 最有趣的是，這些資料並不是來自單一來源。 開發者把大量公開資料整合到同一個介面裡，讓原本散落在各處的資訊，可以像看 Google Maps 一樣直接探索。 除了地圖之外，它還內建不少 OSINT（開源情報）工具： • WHOIS 查詢 • DNS 分析 • Port Scanner • IP 情報查詢 • 漏洞掃描工具 很多原本需要切換好幾個網站才能完成的工作，現在可以直接在同一套系統裡完成。 更誇張的是： 它完全開源。 用 Docker 幾分鐘就能在自己的電腦跑起來，而且大部分功能連 API Key 都不用申請。
