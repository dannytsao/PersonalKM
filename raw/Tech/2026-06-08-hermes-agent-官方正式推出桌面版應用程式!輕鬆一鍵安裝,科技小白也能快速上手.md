# Hermes Agent 官方正式推出桌面版應用程式！輕鬆一鍵安裝，科技小白也能快速上手

## 摘要
Hermes Agent 官方正式推出桌面版應用程式，提供一鍵安裝功能，讓科技小白也能輕鬆快速上手。過去需要透過 CLI 介面安裝和設定的繁瑣步驟，現在變得更加簡單方便。
