## 原始貼文
截至 2026 年 6 月，智譜 AI（Z.ai）針對旗艦模型 GLM-5.2 提供了非常靈活的收費模式，主要分為 API 按量計費（Pay-as-you-go）、GLM Coding Plan 訂閱制，以及第三方平台聚合方案。其整體價格大約只有 GPT-5.5 或 Claude 4.8 Opus 的六分之一，性價比極高。 [1, 2, 3, 4] 
以下為您整理三大主流的付費管道與詳細費率：
## 1. 官方 API 按量計費（最適合開發者、企業整合）
不論是在[智譜 AI 開放平台 (bigmodel.cn)](https://bigmodel.cn/pricing) 還是海外的 [Z.ai 官方開發者文件](https://docs.z.ai/guides/overview/pricing)，GLM-5.2 的基礎定價皆保持一致（僅因幣種換算微幅不同）：

| 計費項目 [1, 5] | 人民幣價格 (每百萬 Token) | 美元價格 (每百萬 Token) |
|---|---|---|
| 輸入 (Input) | ¥ 8.00 | $ 1.40 |
| 輸出 (Output) | ¥ 28.00 | $ 4.40 |
| 快取命中 (Cache Read) | ¥ 2.00（大幅節省重複輸入成本） | $ 0.26 |
| 快取存儲 (Cache Storage) | 限時免費 | 限時免費 |

------------------------------
## 2. GLM Coding Plan 訂閱制（最適合工程師、重度代碼用戶）
如果您打算將 GLM-5.2 用在 IDE 插件（如 Cursor、Cline 或 [Claude Code](https://www.developersdigest.tech/blog/glm-5-2-free-and-cheap-access-2026)）中進行高強度的代碼編寫，官方推出了 GLM Coding Plan 程式碼訂閱套餐，不需手動綁定 API 扣款：

* Lite 輕量版：約 $10 – $12.60 / 月，提供基本的輕量迭代額度。
* Pro 專業版：約 $30 / 月，提供更高的權重與調用額度。
* 計費係數注意：由於 GLM-5.2 屬於前沿推理模型，在訂閱制中，每日 14:00～18:00（高峰期）將按 3 倍係數扣除套餐額度，非高峰期按 2 倍扣除（限時福利：2026 年 9 月底前非高峰期僅按 1 倍抵扣）。 [2, 3] 

------------------------------
## 3. 第三方平台與聚合 API 價格
如果您已經習慣使用模型路由平台，也可以透過以下渠道接入：

* [OpenRouter](https://openrouter.ai/z-ai/glm-5.2)：輸入 $1.40 / M，輸出 $4.40 / M（部分經由 fp4 量化路由最低可達輸入 $1.20 左右）。
* [硅基流動 (SiliconFlow)](https://siliconflow.cn/pricing)：輸入 ¥ 8.00 / M，輸出 ¥ 28.00 / M，快取價格 ¥ 2.00 / M。
* OpenCode Go 聚合訂閱：每月 $10（首月優惠 $5），打包了一定額度供用戶在 OpenClaw 等自動化代理工具中調用 GLM-5.2。 [6, 7, 8, 9, 10] 

------------------------------
## 4. 0 元體驗管道（薅羊毛）

* 本地自託管：因為 GLM-5.2 的模型權重是基於 MIT 協議完全開源，如果您本地有足夠的硬體資源，可以直接透過 [Ollama](https://ollama.com/library/glm-5.2) 免費下載部署，完全不需要支付任何 Token 費用。

如果您需要進一步的資訊，可以告訴我：

* 您打算將 GLM-5.2 應用在 代碼編寫 (Coding)、自動化 Agent 還是 常規聊天 中？
* 您需要我提供在 Cursor 或 VS Code 中配置該 API 的教學 嗎？
