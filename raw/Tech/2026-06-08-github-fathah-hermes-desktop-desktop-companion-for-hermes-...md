Hermes Desktop 是一款由 Nous Research 推出的官方原生圖形介面（GUI）應用程式，專為管理與執行具備「自我進化能力」的開源 AI 助理 [Hermes Agent](https://github.com/nousresearch/hermes-agent) 所設計。它在 2026 年 6 月推出公測版（Public Preview），讓使用者不再需要透過複雜的終端機（CLI）指令，就能直接在 Windows、macOS 與 Linux 系統上輕鬆操作 AI Agent。 [1, 2, 3] 
## 核心功能優勢

* 一鍵式引導安裝：自動執行背景安裝腳本，並將資料統一儲存於 ~/.hermes 目錄下。
* 閉環學習與記憶：擁有「持久記憶（Persistent Memory）」與「自我進化學習圈」，能從成功的任務中自動提煉並優化技能。
* 多模型自由切換：整合 [Nous Portal](https://www.facebook.com/kocpctw/posts/hermes-agent-%E6%A1%8C%E9%9D%A2%E7%89%88%E6%AD%A3%E5%BC%8F%E6%8E%A8%E5%87%BA%E5%8E%9F%E6%9C%AC%E8%A6%81%E9%9D%A0-cli-%E5%AE%89%E8%A3%9D%E8%A8%AD%E5%AE%9A%E7%8F%BE%E5%9C%A8-windowsmacoslinux-%E9%83%BD%E8%83%BD%E4%B8%80%E9%8D%B5%E4%B8%8A%E6%89%8B%E6%83%B3%E8%A9%A6-ai-agent-%E7%9A%84%E4%BA%BA%E9%96%80%E6%AA%BB%E4%BD%8E%E5%BE%88%E5%A4%9A/1579783313517697/)、OpenRouter、OpenAI 及本地的 Ollama，支援超過 300 種大型語言模型。
* 跨通訊軟體控制：串聯 Telegram、Discord、Slack、WhatsApp、Signal 等 16 種即時通訊平台，且各平台間保有「統一記憶與對話脈絡」。
* 任務並行子代理：支援自然語言排程（Cron 任務），並能派發複雜工作給多個「子代理（Sub-agents）」進行平行處理。
* 五種沙盒安全環境：提供本機、Docker、SSH、Singularity 及 Modal 等後端隔離選項，確保自動化執行時的電腦系統安全。 [1, 4, 5, 6, 7, 8] 

## 開源與下載資訊
Hermes Desktop 採用 MIT 授權開源發布，使用者可以至 Hermes Agent 官方網站 下載各平台的安裝檔（如 Windows 的 .exe 或 Mac 的 .dmg）。若電腦中已安裝過舊版 CLI 引擎，桌面版會自動偵測並無縫繼承既有的設定、記憶與 API 金鑰，不會造成資料衝突。 [7, 9, 10, 11, 12, 13]
