# 大部分的 RAG 都是個黑箱。 | Facebook

## 摘要
本文討論了大部分的 RAG（檢索增強生成模型）運作如同黑箱，使用者無法看到其檢索過程和依據。作者介紹了 KnowDB，一個讓 Agent 以導航方式而非僅查詢的知識庫，並且將其導航過程可視化，讓使用者能看到 Agent 如何逐步讀取文件內容。
