## 摘要
RC Astro 推出獨立運行的 RC-Astro 工具，包含 BlurXTerminator、NoiseXTerminator 和 StarXTerminator，無需依賴 PixInsight 或 Photoshop。這些工具支援 Windows、macOS 和 Linux 平台，並擴展了 GPU 加速支援，包括 AMD 和 Intel GPU。用戶可免費下載 CLI 版本，並可輕鬆整合至其他應用程式中。

Stand-alone, independent of any particular host application Easily integrated into any application, or use it from a terminal Works on Windows, macOS, and Linux Expands Windows GPU support to AMD and Intel Existing licenses honored (no additional cost) Free to download Why is this important? Until now, StarXTerminator and NoiseXTerminator required PixInsight or Photoshop, and BlurXTerminator was tied exclusively to PixInsight due to deconvolution requiring linear (un-stretched) data. With this release, the requirement for these host applications is eliminated. Possibly even more importantly, this can be easily integrated into any host application that has the ability to interface with an external tool. The CLI was designed to make it easy for developers to integrate. In fact, both Siril and Seti Astro Suite Pro have already done this, and it took them very little time. Siril Integration Seti Astro Pro Suite Integration What is a CLI? A “CLI” is a program with a simple command-line interface instead of a graphical interface. It is invoked from a command or terminal prompt, or by another application. By entering a few simple options, it will load an image file, process it using the specified product with the chosen settings, and write the result to a new file. For example, here is an image being deconvolved by BlurXTerminator: > rc-astro bxt image.tif --sharpen-stars 0.5 --sharpen-nonstellar 0.5 BlurXTerminator, AI version 4, permanently licensed.
