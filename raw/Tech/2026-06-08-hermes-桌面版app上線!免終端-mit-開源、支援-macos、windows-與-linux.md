# Hermes 桌面版APP上線！免終端 MIT 開源、支援 macOS、Windows 與 Linux

## 摘要
Hermes 桌面版APP正式發布，採用MIT授權免費開源，支援macOS、Windows與Linux平台。該應用整合持久記憶、自然語言排程及超過300種模型，提升多平台即時通訊與自動化AI代理的使用便利性，無需依賴終端機操作。
