⚙️  HOW IT WORKS:

REAL-TIME (Every note capture via LINE):
  1. User sends URL via LINE
  2. Bot captures → creates note
  3. ✨ AI enrichment: tags + summary
  4. 🔍 Decay check: analyze freshness
  5. If outdated: Add 🔴 deprecation notice
  6. Commit to GitHub

MONTHLY (1st of month, 9 AM):
  1. Auto-triggered cron job
  2. Scans entire vault
  3. Categorizes by freshness:
     • CRITICAL (6+ mo) 🔴
     • HIGH (4-6 mo) 🟠
     • MEDIUM (3-4 mo) 🟡
     • EVERGREEN (0-3 mo) 🟢
  4. Generates .decay-report-monthly.md
  5. Auto-commits to GitHub
  6. You review priorities
