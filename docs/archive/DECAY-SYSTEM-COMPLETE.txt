⚙️  HOW IT WORKS:

Real-time (Every note capture):
  1. User sends LINE message → Bot captures
  2. Enrichment: Add tags + summary ✨
  3. Decay check: Is this DevOps/AI?
  4. If yes: Detect versions, analyze freshness
  5. If outdated: Add 🔴 deprecation notice
  6. Commit to GitHub

Monthly (1st of month, 9 AM):
  1. Cron job triggers automatically
  2. Scans all DevOps/AI notes in vault
  3. Calculates freshness scores
  4. AI analyzes each for staleness
  5. Generates categorized report
  6. Creates .decay-report-monthly.md
  7. Commits to GitHub
  8. You get monthly summary
