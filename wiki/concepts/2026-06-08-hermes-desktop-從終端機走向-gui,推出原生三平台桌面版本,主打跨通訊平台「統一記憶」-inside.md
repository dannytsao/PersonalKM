# Hermes Desktop 從終端機走向 GUI，推出原生三平台桌面版本，主打跨通訊平台「統一記憶」 - INSIDE

## 摘要
Hermes Desktop 推出原生 GUI 桌面版本，支援 Windows、macOS 及 Linux 三大平台。其特色功能「統一記憶」可跨通訊軟體同步對話脈絡，提升使用體驗並降低使用門檻。
