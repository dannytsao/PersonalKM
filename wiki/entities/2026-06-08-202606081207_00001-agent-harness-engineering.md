# Agent Harness Engineering

## Log ID
202606081207_00001

## 摘要
Agent Harness Engineering 是指將編碼代理模型及其周邊構建視為一個整體，並將其支架工程化，持續改進以減少代理出錯的情況。這種方法強調將代理的支架視為一個真實的工件，並不斷優化。
