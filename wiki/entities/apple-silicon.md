# Apple Silicon

Apple has published an official tutorial showing developers how to run local AI models on Mac to build AI Agents for iPad app development, requiring no API keys or subscriptions. The approach leverages Apple Silicon hardware to execute the full AI loop entirely on-device, ensuring data remains local

## Key Facts

- Apple released an official tutorial for running local AI models on Mac without APIs or subscriptions
- Three-step process targets AI Agent development for iPad apps
- Full inference loop runs locally on Apple Silicon, keeping data on-device
- No usage costs and offline availability enabled by local execution
